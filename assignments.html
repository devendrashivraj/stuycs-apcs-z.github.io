Assignments

  • git pull the homework repository to make sure it's up to date.
  • Add your homework file or folder in your personal homework folder.
  • git add the file, then git commit, then git push
  • Each homework should be named #-whateveryouwant where # is the assignment number. So, if homework 4 was about loops, you might name it 4-loops
  • If the assignment requires a file, just add a file using the above naming convension.
  • If it requires a number of files (or you just prefer to use a folder), make a folder using the above naming convension and add files into that folder (don't forget to git add each one). In this case, you can name the files anything you want.
